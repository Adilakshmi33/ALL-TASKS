1. semantic elements

Element that describes about the content are called semantic elements.
Ex: <p>, <header>, <strong>, <em>

2. Non semantic elements

Elements that doesn't describe about the content are called non semantic elements.
Ex: <i>, <b>

3. Block-level elements

A block level elements always starts in a new line is called block-level element.
Ex: All the tags like <header>, <p>, etc.

4. inline elements

It will just take content space
Ex: <b>, <br>
