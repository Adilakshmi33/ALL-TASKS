SELECTORS

1. Element selector
An element selector in CSS targets and styles HTML elements based on their tag names.
2. Id selector
An ID selector in CSS targets and styles a specific HTML element based on its unique identifier.
3. Class selector
A class selector in CSS allows the styling of multiple HTML elements with a shared class.
4. Group selector
A group selector in CSS lets apply the same styles to multiple selectors by separating them with commas.
5. Universal selector
A universal selector in CSS (* asterisk) selects and styles every element on a webpage.
6. Element.class selector
An element.class selector in CSS (e.g p.highlight) targets and styles specific HTML elements with a particular tag and class combination.
7. Combinator selector Combinator selectors are four types
They are :

  • Descendant selector
  • Child selector
  • Adjacent sibiling
  • General sibiling
* Descendant selectors
A descendant selector in CSS selects and styles elements that are inside a specified parent element, regardless of their direct or indirect relationship.
* Child selector
A child selector in CSS selects and styles elements that are immediate children of a specified parent element, denoted by the ">" symbol.
* Adjacent sibiling The adjacent sibling combinator in CSS (+) selects and styles an element that is immediately preceded by a specified element with the same parent.
* General sibiling The general sibling combinator (~) in CSS selects and styles all elements that are siblings and come after a specified element, sharing the same parent.
8. Attribute selector
An attribute selector in CSS targets and styles HTML elements based on the presence or value of their attributes.
