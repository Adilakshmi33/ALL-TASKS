1. Tags :
What ever we write in angular brackets is called tags.
2. Elements :
Contents enclosed between open and closing tag is called elements.
3. Self-Closing tags :
Self-Closing tags doesn't need to be closed manually bt its closing tag, All the self-Closing tags are void elements.
4. Comments :
Comments is used for only understand the code easily.
5. Semantic elements :
Element that describes about the content are called semantic elements.
EX:
  • <p>
  • <header>
  • <strong>
  • <em>
6. Non Semantic elements :
Element that doesn't describes about the content are called non semantic elements.
EX:
  • <i>
  • <b>
7. Block-level elements :
A block level elements always starts in a new line is called block-level element.
EX: All the tags like,
  • <p>
  • <header>, etc
8. inline elements :
It will just take content space.
EX:
  • <b>
  • <br>
