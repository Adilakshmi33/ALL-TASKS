html concepts

1. Tags

Whatever we write in angular brackets is called tag.

2. Elements

Content enclosed between open and closing tag is called elements.

3. Self-Closing tag

It doesn't need to be closed manually by its closing tag, i.e it doesn't have a separate closing tag.

4. Blocklevel elements

A block-level element always starts on a new line.

5. Comments

Comments is used to understand the code whatever we are written.
