Difference between Body element selector & universal selector

1. Body element selector:

  • The body selector is used to target the element of an HTML document.
  • It is written as body in CSS.
  • Styles applied with the body selector will affect the entire body of the HTML document. 
      body { 
             /* Styles specific to the body element go here */ 
      }

2. Universal selector
  • The universal selector is denoted by an asterisk * and is used to select all elements on a page.
  • It applies styles to every element in the HTML document. 
      *{ 
         /* Styles that apply to all elements go here */ 
      }
